\# Sentinel - Linux System Auditor

A modular Python-based hardware diagnostic and security logging tool.



\## Features

\- \*\*Hardware Monitoring:\*\* Tracks CPU usage, RAM availability, and Disk space using Linux kernel utilities (`top`, `free`, `df`).

\- \*\*Security Auditing:\*\* Logs system access events and user session history to an encrypted-ready `.log` file.
